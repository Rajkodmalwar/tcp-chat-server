# Chat Server Demo Screenshots

Add your screenshot images here:

- `client_alice.png` - Alice's client terminal
- `client_bob.png` - Bob's client terminal  
- `client_charlie.png` - Charlie's client terminal

## How to add screenshots:

1. Take screenshots of each terminal showing the chat in action
2. Save them to this folder
3. Commit and push:
   ```bash
   git add screenshots/
   git commit -m "Add demo screenshots"
   git push
   ```
